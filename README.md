# game_project

![CMake](https://github.com/Pierre-Genest/game_project/workflows/C++%20CMake%20Build/badge.svg)

KMU fall's semester game project

## Build and Run

```sh
# Unix command

# Generate the cmake project
$> ./tools/generate.sh

# Build all the target
$> ./tools/build.sh

# Run the executable
$> ./build/src/Application/app
```
